XQueryEvaluator

XQueryEvaluator is a little editor to help writing XQuery scripts. It features highlighting and basic auto completion and error handling. Basic XSLT and XPath support is also included.

Dependencies

Qt 4.5 or later

Author

Nils Fenner

Download

You can download the source in either zip or tar formats. If you like, you can also clone the project with Git by running: 

$ git clone git://github.com/antis81/XQueryEvaluator

get the source code on GitHub : antis81/XQueryEvaluator
